本框架为人工智能领域框架，使用Java来实现。项目编码为GBK，JDK版本为1.6。项目使用eclipse开发，所以下载后直接导入eclipse即可。

 神经网络构建代码

// 构建神经网络
//    1. 构建神经元
Neural neural1 = new Neural(0.25);
Neural neural2 = new Neural(-0.75);
Neural neural3 = new Neural(-0.45);
//    2. 构建神经层
Layer inputLayer = new Layer();
inputLayer.add(neural1);
inputLayer.add(neural2);
inputLayer.add(neural3);
Layer hiddenLayer = new Layer(256);
Layer hiddenLayer1 = new Layer(256);
Layer outputLayer = new Layer(1);

神经元构造器的输入参数为具体值，二神经层的输入参数为神经元个数。这两个组件创建完毕，则神经网络骨架已经建立完成。对于具体的神经网络构建，则根据方法（如：前溃算法、反向传播算法等）的输入参数来作相应的组件封装。多啰嗦一句，当特征向量维数很大时，则将特征向量存入List对象中，通过遍历每一个值赋值到神经元中。
